1., spring run hello.groovy -- --server.port=3333
	A spring cli elindit egy Tomcat server-t és a 3333 port-on futtatja a groovy fájlt.
2., java -Dserver.port=3333 -jar target/ElsoJava-0.0.1-SNAPSHOT.jar 
Mielott ezt futtatom nyomtam egy mvn clean install-t, azért hogy elkeszuljon a jar fájlom. Aztán futtatom a jar-t ezzel a paranccsal.
